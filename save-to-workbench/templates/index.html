Save to Workbench Flow POC

Method 1: Direct GCS URIs

Import files directly from Google Cloud Storage using gs:// URIs. Files must be publicly accessible or accessible to your Workbench account.



Method 2: TSV Manifest URL

Import multiple files using a TSV manifest file. The manifest URL must be publicly accessible, but can contain signed URLs to private GCS resources.



Method 3: File Upload with Auto-Generated Manifest

Upload a file which will be stored in GCS, then automatically create a manifest with signed URL and import to Workbench.
